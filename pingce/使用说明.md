## 🔗 后续如何使用 RAGAS 评测？

运行完批量测试后，创建一个新的 Python 文件 `run_ragas_eval.py`：

```python
import pandas as pd
import json
from datasets import Dataset
from ragas import evaluate
from ragas.metrics import faithfulness, answer_relevancy, context_precision

# 读取批量测试结果
df = pd.read_excel("batch_test_results.xlsx")

# 解析 contexts 列（JSON 字符串转列表）
df['contexts'] = df['contexts'].apply(lambda x: json.loads(x) if isinstance(x, str) else x)

# 创建 RAGAS 数据集
dataset = Dataset.from_dict({
    'question': df['question'].tolist(),
    'answer': df['answer'].tolist(),
    'contexts': df['contexts'].tolist(),
    'ground_truth': df['ground_truth'].tolist()
})

# 执行评测
print("开始 RAGAS 评测...")
result = evaluate(
    dataset=dataset,
    metrics=[faithfulness, answer_relevancy, context_precision]
)

print("\n评测结果：")
print(result)

# 保存结果
df_result = result.to_pandas()
df_result.to_excel("ragas_evaluation_results.xlsx", index=False)
print("\n✅ RAGAS 评测结果已保存到: ragas_evaluation_results.xlsx")
```

然后运行：
```bash
python run_ragas_eval.py
```

## ❓ 常见问题

### Q1: 提示找不到 `datasets` 模块？

安装依赖：
```bash
pip install datasets pandas openpyxl
```

### Q2: 提示知识库不存在？

检查 `KNOWLEDGE_SPACE` 是否与你创建的知识库名称完全一致。

### Q3: 运行很慢怎么办？

这是正常的，因为要逐个调用 LLM API。脚本每处理 10 个问题会自动保存一次，即使中断也不会丢失数据。

### Q4: 可以中途停止吗？

可以！按 `Ctrl+C` 停止。已处理的结果已经保存在 `batch_test_results.xlsx` 中。
